## 1️⃣ PR 标题规范
> 使用英文

[UI] Feature: Modify the username and display name
[Design-System] Update: Button disabled state
[Template] Update: Sprite config

## 2️⃣ PR 描述必须包含
#### 背景
为什么改？

#### 修改内容
- 调整按钮 padding
- 修改错误提示颜色

#### 影响范围
- 产品登录页
- 主要按钮

#### 是否影响设计系统
- [x] 是（需要 design system PR）
- [ ] 否
